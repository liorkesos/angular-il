הערת המפתח

ברוכים הבאים. וורדפרס היא פרוייקט מיוחד מאוד בשבילי. כל המפתחים והתורמים האחרים מוסיפים משהו למערכת, וביחד אנחנו יוצרים משהו יפהפה שאני גאה להיות חלק בו. אלפי שעות הושקעו בוורדפרס, ואנחנו ממשיכים לשפר אותה מדי יום. תודה לך על שהכנסת אותה לעולם שלך.

מאט מולנווג

התקנה - חמש דקות קצרות

ברוב המקרים, ההתקנה של וורדפרס יכולה להתבצע כולה דרך הדפדפן. אם פעולות כמו עריכת קובצי PHP נשמעות לכם מדאיגות, פשוט נסו להעלות את כל הקבצים של וורדפרס לשרת ולגשת לכתובת שלה בדפדפן. עם זאת, זכרו שלא תוכלו להתחמק מדברים כאלה לנצח...

  1. פתחו את הקובץ שהורדתם (wordpress.zip או wordpress.tgz) בתיקיה ריקה.
  2. פתחו את הקובץ wp-config-sample.php בעורך טקסט (כמו Notepad, לא כמו Word), ומלאו את פרטי החיבור לבסיס הנתונים, לפי ההוראות.
  3. שמרו את הקובץ בשם ‎wp-config.php‏.
  4. העלו את הכל לאתר.
  5. הפנו את הדפדפן לכתובת ‎/wp-admin/install.php‏ - תוכנית ההתקנה תלווה אתכם עם ההוראות המתאימות. אם ההתקנה נכשלת, בדקו את את ההגדרות בקובץ wp-config.php‏. אם ההתקנה עדיין נכשלת, פנו לפורומים עם נתונים רבים ככל האפשר אודות ההתקנה.
  6. רשמו את הסיסמה במקום בטוח, או בכל אופן זכרו אותה בצורה כלשהיא.
  7. לחצו על הקישור לעמוד הכניסה. הכניסו את שם הכניסה 'admin' ואת הסיסמה שקיבלתם בעמוד הקודם.
  8. סיימתם את ההתקנה, ואתם נמצאים עתה בממשק הניהול של הבלוג שלכם. מזל טוב!

הוראות התקנה מלאות בוויקי של וורדפרס בעברית

שדרוג - שלושה שלבים קלים

לפני השדרוג, הכינו גיבויים של בסיס הנתונים וכן של כל הקבצים בתיקיה של וורדפרס בעברית.

  1. צרו גיבוי של כל התיקה של וורדפרס, וגבו גם את בסיס הנתונים.
  2. מחקו את הקבצים של ההתקנה הקיימת של וורדפרס (מלבד הקבצים שאתם העלתם או קבצים שעשיתם בהם שינויים) והעלו במקומם את הקבצים של הגרסה החדשה
  3. פתחו את העמוד ‎ /wp-admin/upgrade.php ‏ בדפדפן.
